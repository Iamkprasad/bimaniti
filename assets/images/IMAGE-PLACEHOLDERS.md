# Image Placeholder Reference — AI Image Prompts

All images in this project are **placeholders** — generic `.jpg` files (mostly solid colour backgrounds). Below is every image needed by the site, with **text-to-image prompts** you can paste directly into Google Gemini, Imagen, or any AI image generator.

**Rules for these prompts:**
- No company logos, no brand names, no trademarks
- Describe **scenes, concepts, compositions** — not specific buildings or products
- Use "editorial illustration style" or "editorial photograph style" as the consistent aesthetic

## How to use these prompts

1. **Google Gemini** — Paste the prompt directly. Ask for a "photorealistic editorial photograph" or "vector editorial illustration" depending on the prompt prefix.
2. **Adobe Firefly / Midjourney / DALL-E** — Same prompts work well. Add `--ar 16:9` or `--ar 4:5` as needed.
3. **Saving** — Overwrite the existing `.jpg` file with the exact same filename in the same directory.
4. **Dimensions target** — Blog/News: 1200×630 px. Stack: 1200×700 px or wider.
5. **File size** — Compress to under 200 KB via squoosh.app or similar.
